# E-Commerce Sales EDA: Decode Lab Internship

Exploratory Data Analysis of 1200 e-commerce orders using Microsoft Excel to identify payment behavior patterns and marketing channel ROI.

##  Dataset Overview
- **Total Records:** 1200 orders
- **Total Revenue:** $1,264,759 
- **Average Order Value:** $1,047
- **Columns:** OrderID, CustomerID, OrderDate, ProductName, Category, Quantity, UnitPrice, TotalPrice, PaymentMethod, ReferralSource, OrderStatus

##  Key Findings
1. **Payment Behavior:** Credit Card highest AOV $1,127. Cash contributes 20.5% orders with $1,056 AOV
2. **Marketing Channels:** Instagram drives 60 Online payments. Email users prefer Debit Card - 59 orders  
3. **Price Sensitivity:** R² = 0.03 shows no correlation between UnitPrice and Quantity
4. **Data Quality:** Outliers detected. Max order $5,790, Min $10

## 🛠️ Tools Used
Microsoft Excel, PivotTables, PivotCharts, Descriptive Statistics, Cross-Tabulation

##  Excel Sheets Included
1. Raw Data - Complete dataset
2. Descriptive Statistics - Mean, Median, Mode, Skewness
3. Outlier Analysis - Box plots and IQR method
4. ReferralSource Analysis - Revenue by marketing channel
5. Payment Analysis - PaymentMethod × ReferralSource cross-tab

##  Business Impact
- Checkout optimization by traffic source recommended
- Cash on Delivery essential - $259K revenue segment
- Cross-sell strategies over discount campaigns advised
